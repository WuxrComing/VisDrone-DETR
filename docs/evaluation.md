# 评估协议与汇总

## 评估器配置

所有模型使用统一的 **VisdroneCocoEvaluator**（COCO 指标适配 VisDrone）：

| 参数 | 值 |
|------|-----|
| `maxDets` | `[1, 10, 100, 500]` |
| areaRng | `small (<32²)`, `medium (32²–96²)`, `large (>96²)` |
| 忽略区域过滤 | IoF > 0.5（遮挡/拥挤区域） |
| 统计量 | 13 个 COCO-style stats |

> **注意**: DQ-DETR 旧版使用 AI-TOD CocoEvaluator（maxDets=[1,100,1500], areaRng=verytiny/tiny/small/medium, 无 ignore 过滤）。新旧评估器面积标签名称相同但范围不同，不可跨评估器直接对比。
